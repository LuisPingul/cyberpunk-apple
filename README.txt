Project Description:
This is a landing page for "Apple Neural," a fictional cyberpunk-themed neural interface product. It features a responsive design, neon aesthetics, and interactive JavaScript elements like mobile navigation and alert pop-ups.

External Resources Used:
- Fonts: "Orbitron" and "Rajdhani" from Google Fonts.
- Images: Stock photography from Pexels.com (Science/Tech category).
- Icons: Custom CSS styling used for feature icons.

Special Features:
- Mobile-responsive navigation menu with JavaScript toggle.
- "Cyberpunk Alert" system: JavaScript event listeners on buttons trigger thematic pop-up messages.
- CSS Grid used for the "Specs" section layout.
- Flexbox used for Navigation and About sections.
